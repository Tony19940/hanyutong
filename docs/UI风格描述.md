请为一个移动端中文学习产品设计 UI，产品名叫“汉语通”，目标用户是柬埔寨中文初学者。产品主要功能包括：滑卡背单词、轻量测验、AI语音对话练口语、个人学习主页。

我希望整体视觉风格深度贴合 Khmer / Cambodian cultural identity，但必须是现代、克制、高级的表达，不要做成传统旅游海报，也不要做成民俗拼贴。

设计目标：
- 面向年轻用户，友好、精致、可信、有文化识别度
- 一眼能感受到柬埔寨文化气质，但仍然像一款现代国际化 App
- 重点是实用学习，不是娱乐化社交产品
- 界面尽量简洁，减少废话，突出操作和内容本身

视觉方向：
- 配色请重点探索：金色、深绿色、皇家蓝、温暖米白
- 可加入少量高棉文化启发的装饰元素，但要抽象化、现代化
- 灵感可以来自：Angkor Wat 建筑层级、Khmer temple ornament、lotus、stone carving rhythm、krama 几何纹理
- 不要使用过多复杂花纹，不要满屏装饰
- 不要卡通幼稚，不要廉价渐变，不要过度游戏化
- 希望有文化感、仪式感和高级感

界面原则：
- mobile-first
- 适合 Telegram Mini App
- 主按钮醒目，卡片圆角大，层级清晰
- 标题和说明文案都尽量短
- 界面应更像高品质学习工具，而不是营销页

请先输出完整的视觉系统方向，然后为以下页面分别设计高保真界面：
1. 登录页 / 激活页
2. 学习首页（滑卡背词）
3. 测验页
4. 对话页（像 Telegram 聊天界面）
5. 我的页面

特别说明：
- 对话页中的 AI 老师是一个真实存在感很强的老师角色，不要做成机器人风格
- 对话页需要像聊天软件一样自然
- 学习页和测验页要统一视觉语言
- 我的页面要简洁，不要堆很多设置项

请给出 2 到 3 套不同但都符合 Khmer-inspired modern design 的方向，我再选一套继续细化。
