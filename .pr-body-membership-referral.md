## What changed
- replaced activation-code-first entry with a 3-day trial + month-card redemption flow
- added membership-aware access control with a free learning layer after expiry
- added invite links, invite stats, and first-paid referral rewards
- upgraded admin key management from permanent activation to dated, extendable month-card keys
- added Spec Kit artifacts for 003-membership-referral

## Why
- users can now see and try the product before paying
- premium value is anchored around AI dialogue and quiz instead of blocking the whole app at launch
- referral mechanics now create a usable growth loop instead of relying on the old share-poster-only flow
- offline Telegram payment can keep working while supporting recurring month-card operations

## User impact
- new users can start a free trial immediately
- expired users keep word-card learning and can renew from inside the app
- Me now shows membership expiry and invite performance
- admin can generate, extend, expire, and reuse bound keys correctly

## Validation
- npm test
- npm run build
